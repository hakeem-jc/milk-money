# Milk Money 🛒

A mobile-first grocery list app built with **Next.js, TypeScript, Tailwind v4, and React Hook Form**.
---

## ✨ Overview

Milk Money is a grocery planning app that will help users:

* Track grocery prices
* Compare estimated vs actual spend
* Optimize shopping decisions

Features

* A checklist of grocery items
* Price display per item
* Running totals
* A modal form to add new items

---

## 🧱 Tech Stack

* **Next.js (App Router)**
* **TypeScript**
* **Tailwind CSS v4**
* **React Hook Form**

---

## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the development server

```bash
npm run dev
```

Visit:

```
http://localhost:3000
```

## 📁 Project Structure

```
app/
 ├── layout.tsx
 ├── page.tsx
 └── globals.css
```

* `layout.tsx` – App metadata + Roboto font
